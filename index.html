Talks & Writing

StepFlow

  • Langflow on StepFlow — Oct 2025. Proof-of-concept running existing Langflow workflows unmodified on StepFlow with automatic parallelization and cross-machine execution — the concrete validation of StepFlow's "drop-in runtime for any GenAI framework" thesis.
  • Announcing StepFlow — Aug 2025. Public launch of StepFlow as an open protocol and runtime for durable, distributed, multi-language GenAI workflows; co-authored with Nate McCall.

RAG & Knowledge Graphs

  • Scaling Knowledge Graphs by Eliminating Edges — The New Stack, Jul 2024. Replaces materialized graph edges with per-node link metadata, dropping node-insertion cost from quadratic to linear and making content-centric graph RAG tractable at document-collection scale.
  • Better LLM Integration with Content-Centric Knowledge Graphs — The New Stack, Jun 2024. Challenges the entity-extraction orthodoxy in graph RAG with a simpler content-centric design that defers concept resolution to the LLM at query time — the conceptual foundation for DataStax's Graph RAG library.
  • Extraction Matters Most — Feb 2024. Empirical study showing that document-extraction quality dominates chunking and retrieval tuning as a RAG performance lever — redirecting where practitioners should invest effort.

Kaskada

  • The Next Step in Event Processing — talk at AICamp Seattle.
  • Kaskada at the Global Feature Store Meetup — joint meetup talk with Salesforce.
  • Four Pillars of Real-Time Generative AI — Nov 2023. A framework identifying four capabilities — monitor, decide, act, improve — that LLM agents need to act continuously on streaming events rather than only respond to prompts.
  • Introducing Timelines (Part 1): an Evolution in Stream Processing Abstraction — May 2023. Introduces Timelines as the next step in a stream-processing abstraction lineage from Kafka through Flink — a multiset ordered by time and grouped by entity that makes temporal queries expressible where relational abstractions fall short; co-authored with Therapon Skoteiniotis (Part 2 covers how the query language is built on the abstraction).
  • Announcing Kaskada OSS — Mar 2023. Public release of the Kaskada engine under Apache 2.0, bringing its temporal columnar computation model to the open-source community.
  • Kaskada User Guide — introduction to Timestreams and the Kaskada engine.
